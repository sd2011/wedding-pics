# אתר שיתוף תמונות לחתונה 💍

אתר דיגיטלי לחתונה שמאפשר לאורחים להעלות תמונות ולצפות בגלריה משותפת.

## תכונות

- 📸 העלאת תמונות מרובות
- 🖼️ גלריה משותפת עם כל התמונות
- 📱 קוד QR להעלאה קלה
- ⚡ עדכון בזמן אמת
- 🎨 עיצוב יפה ומודרני
- 📱 מותאם למובייל
- 🇮🇱 תמיכה בעברית

## התקנה

1. התקן את התלויות:
```bash
npm install
```

2. הגדר משתני סביבה (מומלץ):
```bash
npm run setup-env
```
או צור קובץ `.env.local` ידנית (ראה `ENV_SETUP.md`)

3. הפעל את השרת המקומי:
```bash
npm run dev
```

4. פתח את הדפדפן בכתובת:
```
http://localhost:3000
```

## שימוש

### דף הבית
- מציג מידע על החתונה
- קוד QR לסריקה מהירה
- קישורים להעלאת תמונות ולצפייה בגלריה

### העלאת תמונות (`/upload`)
- בחר תמונות מהמכשיר
- העלה מספר תמונות בבת אחת
- התמונות נשמרות אוטומטית

### גלריה (`/gallery`)
- צפה בכל התמונות שהועלו
- לחץ על תמונה לצפייה בגודל מלא
- עדכון אוטומטי כשמועלות תמונות חדשות

## פריסה (Deployment)

**⚠️ חשוב:** האפליקציה תומכת כעת ב-Cloudinary לאחסון תמונות בענן!

### פריסה מהירה עם Vercel + Cloudinary:

1. **התקן תלויות:**
```bash
npm install
```

2. **צור חשבון Cloudinary** (חינמי): https://cloudinary.com

3. **העלה ל-GitHub** והתחבר ל-Vercel

4. **הוסף משתני סביבה ב-Vercel:**
   - `CLOUDINARY_CLOUD_NAME`
   - `CLOUDINARY_API_KEY`
   - `CLOUDINARY_API_SECRET`

📖 **מדריך מפורט:** ראה `DEPLOYMENT.md` או `QUICK_DEPLOY.md`

### ללא Cloudinary (פיתוח מקומי בלבד):
- האפליקציה תעבוד עם אחסון מקומי
- **לא מומלץ לפריסה** - הקבצים יימחקו

## הערות חשובות

- התמונות נשמרות ב-Cloudinary (בפרודקשן) או ב-`public/uploads` (מקומי)
- המטא-דאטה נשמרת ב-`data/photos.json`
- מומלץ להשתמש ב-Cloudinary לפריסה
